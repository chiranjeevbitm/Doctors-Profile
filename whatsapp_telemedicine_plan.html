Technical Integration Plan

Exact tech stack to integrate into doctors-profile-chi.vercel.app

Your site is on Vercel (Next.js/React likely). All integrations below are plug-and-play with Vercel's serverless architecture. Total integration time: 3-4 weeks for a developer.

Frontend (Your Website)

💬

Custom WhatsApp Widget Build from scratch

A floating React component with chat UI, payment step, status polling, and video join. Styled exactly like WhatsApp. No third-party chat SDK needed — full control.

🔔

Browser Push Notifications Web Push API

When doctor approves, patient gets a browser notification even if they've minimized the tab. Uses the Web Notification API — free, no library needed.

⚡

Real-time Status Updates Supabase Realtime / Firebase

Patient sees live status changes (Pending → Approved → Calling) without refreshing. WebSocket subscription updates the widget in real-time.

Backend & Database

🗄️

Supabase (Free Tier) Database + Auth + Storage

Stores consultations, payment screenshots, prescriptions. Row-level security ensures HIPAA-style data privacy. Realtime subscriptions built-in. Free up to 500MB.

⚙️

Vercel Serverless Functions Already on your platform

API routes for: receiving payment submissions, verifying UPI (Razorpay API), notifying doctor, creating video rooms. No separate server needed.

Video Calling

📹

Daily.co or 100ms.live Recommended — Free tier available

Embed a video call room directly in the chat widget. Daily.co: 10,000 free participant-minutes/month. 100ms: 10,000 mins/month free. Both have React SDKs.

🔗

Alternative: Jitsi Meet (Self-hosted) 100% Free

Open-source video. Host on DigitalOcean ($6/month). No per-minute charges. Good for scale. IFrame embed works cleanly inside the widget.

Payment Verification

💳

Razorpay Payment Links Easiest to start with

Generate a ₹199 payment link. Patient pays and gets auto-redirected back. Razorpay webhook confirms payment to your backend — no manual screenshot verification needed eventually.

📸

Phase 1: Manual Screenshot Upload Quick to implement

Patient uploads UPI payment screenshot → stored in Supabase Storage → doctor reviews in dashboard. Simple, reliable. Upgrade to automated Razorpay later.

Doctor Dashboard

📱

Progressive Web App (PWA) Dashboard Works on phone as app

A separate /doctor route on your Vercel site, password-protected. Doctor adds it to phone home screen like an app. Receives push notifications, views requests, approves with 1 tap.
